# Project Documentation with GitHub

## รายละเอียดโปรเจกต์

โปรเจกต์นี้จัดทำขึ้นเพื่อศึกษาการใช้งาน Git, GitHub และ Sourcetree ในการจัดการเวอร์ชันโค้ด และ เพื่อความสะดวกสบายการทำงาน

## ฟีเจอร์

- หน้า Login
- กรอก Username
- กรอก Password
- ปุ่ม Login
- ปุ่ม Register

## เทคโนโลยีที่ใช้ในโปรเจค

- HTML5
- CSS3
- Git
- GitHub
- Sourcetree
- markdown

## โครงสร้างโปรเจกต์

```text
Project-Documentation-with-GitHub
│
├── login
│   └── login.html
│   └── markdown.html
│
└── README.md
```

## วิธีใช้งาน

1. Clone Repository
2. เปิดไฟล์ login.html
3. ทดสอบผ่าน Web Browser
