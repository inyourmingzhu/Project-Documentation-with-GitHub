## USER REQUIREMENTS

### CUSTOMER

- สมัครสมาชิกและเข้าสู่ระบบ
- ค้นหาและดูรายละเอียดสินค้า
- เพิ่มสินค้าในตะกร้าและสั่งซื้อสินค้า
- ชำระเงินและติดตามสถานะคำสั่งซื้อ
- ดูประวัติการสั่งซื้อ
- ยื่น ticket support (ขอความช่วยเหลือ/ติดต่อ support)

### STAFF
- ตรวจสอบคำสั่งซื้อของลูกค้า
- อัปเดตสถานะคำสั่งซื้อและการจัดส่ง
- ตรวจสอบข้อมูลการขาย
- ตอบ ticket หรือการรายงานปัญหาของลูกค้า (Customer)

### ADMIN

- จัดการข้อมูลสินค้า (เพิ่ม แก้ไข ลบ)
- จัดการคำสั่งซื้อ
- จัดการข้อมูลผู้ใช้งาน
- กำหนดสิทธิ์การใช้งานของระบบ
